how to set up for dev
1.install os (operation system) -> Ubuntu (winpty docker run -it ubuntu bash)
2.update ubuntu repo apt update (apt update)
3.install python apt install python3 -y
4.install python3-pip (apt install python3-pip -y)
5.install Flask dependencies (pip3 install flask)
6.write python app (app.py)
7.start flask app (FLASK_APP=app.py flask run --host=0.0.0.0)

#Dockerfile
FROM ubuntu

RUN apt update
RUN apt-get install python3 -y
RUN apt-get install python3-pip -y
RUN pip3 install flask

COPY app.py /opt/

ENDTRYPOINT FLASK_APP=/opt/app.py flask run --host=0.0.0.0 --port=8080

##end
